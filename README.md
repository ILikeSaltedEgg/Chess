# 3D Pixel Chess

A minimalist 3D pixel-style chess game with user authentication and game state persistence.

## Features
- 3D pixel-style chess board
- User authentication (signup/login)
- Game state saved automatically
- Turn-based multiplayer logic
- Checkmate detection

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: PHP
- **Database**: MySQL
- **Server**: Apache (via XAMPP)

## Quick Start

### 1. Requirements
- XAMPP (Apache + MySQL + PHP)
- Modern web browser

```bash
# 1. Start XAMPP and run Apache/MySQL
# 2. Place project in: C:\xampp\htdocs\chess\
